# 30-Day Security Plan for Small NGOs

This plan is designed for small teams with limited time and budget.

## Week 1: Protect accounts

- Enable multi-factor authentication for email and admin accounts.
- Identify all shared accounts.
- Move passwords into a password manager.
- Remove unused accounts.
- Make a list of critical systems.

## Week 2: Review access

- Review who has access to shared drives and folders.
- Remove access for people who no longer need it.
- Avoid public sharing links for sensitive documents.
- Identify documents containing personal or sensitive information.

## Week 3: Prepare for common incidents

- Create a compromised account response plan.
- Create a lost device response plan.
- Decide who should be contacted during an incident.
- Save emergency contact details somewhere accessible.

## Week 4: Build basic security habits

- Give staff and volunteers a short security briefing.
- Explain how to report suspicious emails or messages.
- Review device updates and screen locks.
- Set a recurring monthly access review.

## Minimum goal

At the end of 30 days, your organization should have:

- MFA enabled on critical accounts
- no shared passwords outside a password manager
- a basic list of critical systems
- reviewed access to shared documents
- at least one incident response playbook
