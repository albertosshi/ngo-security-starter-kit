# Start Here

This kit is for small NGOs, volunteer groups, community projects, and mission-driven teams that want to improve security without needing a dedicated security team.

You do not need to do everything at once.

Start with these five actions:

1. Enable multi-factor authentication for all important accounts.
2. Stop sharing passwords.
3. Review who has access to shared documents and folders.
4. Make sure someone knows what to do if an account is compromised.
5. Remove access when volunteers, staff, or collaborators leave.

## What to protect first

Focus first on:

- email accounts
- admin accounts
- shared drives and documents
- password managers
- donor, beneficiary, volunteer, or case information
- devices used to access sensitive data

## A useful rule

If losing access to something would stop your organization from operating, protect it first.

If leaking something would harm a person, protect it first.
