# NGO Security Starter Kit

A practical, free security starter kit for small NGOs, volunteer groups, community projects, and mission-driven teams.

Many small organizations protect vulnerable people, coordinate volunteers, collect sensitive information, and depend on email, cloud documents, messaging apps, and shared devices. But they often do not have a dedicated security team.

This project provides simple checklists, playbooks, templates, and guidance to help small teams improve their security step by step.

## Who this is for

This kit is for small organizations that use tools like:

- Google Workspace or Gmail
- Microsoft 365
- Slack, WhatsApp, Signal, or email
- shared cloud documents
- laptops and personal devices
- volunteers, temporary staff, or external collaborators

## What this project helps with

- enabling multi-factor authentication
- reducing account takeover risk
- reviewing access to shared documents
- safely onboarding and offboarding volunteers
- responding to phishing
- responding to compromised accounts
- protecting sensitive information
- creating basic security habits without needing a security department

## What this is not

This is not a replacement for professional legal, compliance, privacy, or security advice.

It is a practical starting point for small teams that need to reduce risk quickly and responsibly.

## Start here

1. Read `docs/start-here.md`
2. Follow the `docs/30-day-security-plan.md`
3. Complete the `checklists/password-and-mfa-checklist.md`
4. Prepare at least one incident response playbook
5. Review access for staff, volunteers, and shared documents

## Project philosophy

Practical over perfect.  
Clear over technical.  
Human safety over compliance theater.

Security should help people do good work safely.
